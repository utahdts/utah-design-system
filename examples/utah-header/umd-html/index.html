Universal Module Definition (UMD) - Using Unpkg CDN

  • This example uses the unpkg CDN to pull in the javascript in with a <script> tag in the <head>.
  • The Utah Header functions are available on the window object: window["@utahdts/utah-design-system-header"]
  • The CSS is included as a standard stylesheet link.
  • The design system colors are overridden.
  • The header is configured in the event utahHeaderLoaded due the asynchronous nature of this example.
  • The header is loaded into a target <div>.
  • The footer is loaded into a <footer> landmark.
  • The footer is loaded after a delay of 2 seconds to illustrate that you can load additional settings (footer) later and it will not clobber the existing settings.
  • Find more information on the Utah Design System Website.
